# DowUrk Sports & Leadership Park — Website

This is the source code for the **DowUrk & Hammond Jr. Tors Community Sports Campus** website.

## Tech Stack
- **Frontend:** React 19 + Tailwind CSS 4 + TypeScript
- **Backend:** Express 4 + tRPC 11
- **Database:** MySQL / TiDB (Drizzle ORM)
- **Auth:** Manus OAuth
